⚡ EV Sales Analysis & Forecasting

📌 Overview

This project focuses on analyzing Electric Vehicle (EV) sales data and building a machine learning model to predict future sales trends. It combines Exploratory Data Analysis (EDA) with time-series aware modeling techniques to extract meaningful insights and improve forecasting accuracy.

🎯 Objectives

Analyze sales trends by state, category, and time

Identify growth patterns and seasonality

Build a predictive ML model

Avoid data leakage using proper time-based validation


📊 Key Insights

EV sales show a strong upward trend

Sales vary significantly across states & vehicle categories

Seasonal patterns observed in monthly data

Rolling averages highlight long-term trends


🧠 Model

Algorithm: Random Forest Regressor

Validation: TimeSeriesSplit (no random split)

Metrics: MAE, RMSE, R²

🛠️ Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, Scikit-learn
